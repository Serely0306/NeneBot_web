# HarukiBot NEO

一款多功能 QQ 群机器人，主要提供《世界计划 多彩舞台》相关查询服务。

本部分的全部教程文档由 HarukiBot 开发者们编写，感谢他们的付出。

### 阅读前提示

**快速开始：** HarukiBot NEO 具有配套功能网站 [Haruki 工具箱](https://neo.haruki.seiunx.com)。

**重要声明：**

- 该 Bot 不提供私聊服务。
- 使用该 Bot，即意味着你同意 [使用条款](https://neo.haruki.seiunx.com/licence/) 及 [隐私条款](https://neo.haruki.seiunx.com/privacy/)。
- 该使用文档或许存在遗漏或者错误内容，如果发现请联系 HarukiBot 分布式 Bot 主反馈。
- 如果你在使用过程中遇到任何问题，你可以在该页面最下方的 [关于](#关于) 下面联系开发者进行反馈。
- 目前 HarukiBot NEO 仍然处于开发状态，有多种功能仍在开发中，如果没有你需要的功能，请不要着急，请耐心等待开发者开发完善。

完整文档和最新功能请访问 [Haruki 官方帮助站点](https://neo.haruki.seiunx.com)。以下为常用指令速查。

## 账号与个人资料

| 指令 | 功能 |
|------|------|
| `/绑定` `/pjsk bind` | 通过游戏UID绑定账号，如 `/绑定 114514` |
| `/个人信息` `/profile` | 生成个人信息图片，可输入UID查询指定账号 |
| `/绑定列表` | 列出当前QQ号所有已绑定的账号 |
| `/设置主账号` `/设置默认绑定` | 设置默认查询的账号 |
| `/交换绑定` | 交换已绑定账号的顺序 |
| `/解绑` `/pjsk unbind` | 通过UID解绑账号 |
| `/隐藏抓包` / `/展示抓包` | 隐藏/展示Suite抓包详细数据 |
| `/隐藏烤森抓包` / `/展示烤森抓包` | 隐藏/展示烤森抓包详细数据 |
| `/隐藏ID` / `/显示ID` | 在查询中隐藏/显示自己的UID |
| `/抓包数据` `/sud` | 查看Suite抓包数据上传时间 |
| `/烤森抓包数据` `/msd` | 查看烤森抓包数据上传时间 |
| `/pjsk验证` | 验证游戏账号（部分功能需验证后使用） |
| `/查时间` `/注册时间` | 查询账号注册时间 |
| `/上传个人信息背景` | 上传自定义个人信息背景图（需验证） |
| `/设置个人信息` `/调整个人信息背景` | 调整背景图：横屏/竖屏、模糊(0-10)、透明度(0-100) |

## 卡牌

| 指令 | 功能 |
|------|------|
| `/查卡` `/pjsk card` | 按属性、角色或ID查卡。支持团名、稀有度、属性、限定类型、技能类型、年份等参数。 |
| `/卡牌列表` `/cards` | 按条件筛选卡牌列表 |
| `/卡牌一览` `/box` `/查箱` | 按条件筛选卡牌，未拥有的以灰色显示（需Suite数据） |
| `/查卡面` `/卡图` | 按ID查询指定卡的卡面原图 |

<details class="cmd-details">
<summary>卡牌查询可选参数</summary>
<div class="cmd-details-content">

- 团名：`ln` `vbs` `ws` `mmj` `25`
- OC/纯VS/VS：`mmjoc` `25oc` `纯v` `mmjv`
- 角色昵称：`miku` `mnr`
- 稀有度：`4` `四星` `生日` `4星`
- 属性：`cool` `蓝` `蓝星`
- 限定类型：`非限` `限定` `期间限定` `fes限定`
- 技能类型：`奶卡` `判` `分` `p分`
- 年份/活动ID/箱活缩写：`2025年` `event123` `mnr1`
- 单卡查询：`miku-1`（倒数第几张卡）

</div>
</details>

## 音乐/乐曲

| 指令 | 功能 |
|------|------|
| `/查曲` `/song` `/music` | 查询单曲信息。支持ID、负数索引、活动ID、别名搜索、模糊匹配。 |
| `/难度排行` `/歌曲列表` `/定数表` | 查询指定难度下的歌曲列表（有Suite数据显示进度） |
| `/谱面预览` `/查谱面` `/技能预览` | 查询指定歌曲的谱面预览 |
| `/打歌奖励` `/歌曲挖矿` | 查询剩余打歌奖励（需Suite数据） |
| `/打歌进度` `/progress` | 查询指定难度Clear/FC/AP完成度（需Suite数据） |
| `/查物量` | 查询指定物量下的歌曲 |
| `/查bpm` | 查询指定BPM下的歌曲 |
| `/查曲绘` `/曲绘` | 查询指定歌曲的曲绘原图 |

## 活动 / 榜线 / 组卡

| 指令 | 功能 |
|------|------|
| `/查活动` `/活动` `/event` | 查询当前或指定活动信息 |
| `/活动列表` `/活动一览` | 活动列表，支持按团/角色/属性/年份/类型筛选 |
| `/冲榜记录` `/活动记录` | 查询账号冲榜记录（需Suite数据） |
| `/sk线` `/榜线` `/skl` | 查询活动榜线 |
| `/sk` `/sk查分` | 查指定分数榜位 |
| `/时速` `/sks` | 查询当前榜线时速。加数字参数转换时间范围的PT增长为时速。 |
| `/日速` `/skds` | 查询榜线日均速度 |
| `/查房` `/cf` | 查询当前房间周回时速等信息 |
| `/玩家追踪` `/ptr` | 查询账号冲榜统计数据 |
| `/档线追踪` `/rtr` `/skt` | 查询档线历史轨迹 |
| `/sk预测` `/榜线预测` `/skp` | 查询榜线预测（33Kit/MoeSekai/SekaRun） |
| `/5v5预测` `/胜率预测` | 查询5v5胜率预测 |
| `/活动组卡` `/组卡` `/配队` | 根据当前活动加成计算组队 |
| `/挑战组卡` | 根据角色计算每日挑战组队 |
| `/最强组卡` `/长草组卡` | 无活动时最强组队 |
| `/加成组卡` `/控分组卡` | 根据指定加成值计算组队 |
| `/烤森组卡` `/ms组卡` | 计算挖烤森获取PT的队伍（国服不开放） |

<details class="cmd-details">
<summary>组卡参数说明</summary>
<div class="cmd-details-content">

- **歌曲和难度**：日服已实装的所有歌曲、所有难度
- **Live类型**：`协力` `单人` `solo` `自动` `auto`
- **组卡目标**：`综合力` `实效`
- **指定体力**：`1火` `5火` `10火`
- **区域道具**：`顶配`（满级）`次顶配`（15级）
- **养成状态**：`禁用` `满破` `满技能` `已读` `画布`
- **技能顺序**：`技能顺序平均` `技能顺序最大` `技能顺序最小`
- **强制包含**：角色名放最后强制为队长，或指定卡牌ID

</div>
</details>
