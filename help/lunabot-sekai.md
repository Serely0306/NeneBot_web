# LunaBot — Sekai 查询

基于 [NeuraXmy/lunabot](https://github.com/NeuraXmy/lunabot) (MIT) 部署，Project Sekai（世界计划）多区服游戏数据查询服务。使用前缀修改为了 `*`(半角符号)。发送 `*help sekai` 查看服务帮助图片，其中标记 🛠️ 的指令仅超级管理员可用。

## 初次使用

- 发送 `*绑定 你的游戏ID` 绑定游戏账号，发送 `*绑定` 查询绑定情况
- 多次绑定可绑定多个账号，最后一个绑定的设为主账号
- 查询指令默认使用主账号，使用其他账号需加 `ux`，例如查询第二个账号：`*个人信息 u2`
- 发送 `*抓包` 获取抓包上传教程（部分功能需要上传抓包数据）

## 多服务器指令

NeneBot处仅部署了日/国服，在指令后加区服缩写指定服务器：`jp`日服 `cn`国服。例如 `*cn查卡` 查询国服卡牌。设置默认区服后可不加前缀。比如你默认绑定了国服账号那么你用 `*查卡` 就等同于 `*cn查卡`

## 账号设置

| 指令 | 功能 |
|------|------|
| `*绑定` `*pjsk bind` | 绑定游戏ID。不加参数查询当前绑定详情；`*绑定 12345` 绑定指定ID。多次绑定可绑定多个账号，最后一个设为主账号。 |
| `*解绑` `*pjsk unbind` | 解绑指定序号的账号，如 `*解绑 2` |
| `*主账号` `*pjsk set main` | 设置主账号为指定序号，如 `*主账号 2` |
| `*交换绑定` `*pjsk swap bind` | 交换两个已绑定账号的顺序，如 `*交换绑定 1 2` |
| `*pjsk隐藏id` `*pjsk展示id` | 隐藏/展示查询结果中的游戏UID。绑定后默认隐藏，每个服务器需分别设置。 |
| `*pjsk服务器` `*pjsk区服` | 设置默认查询区服，不加参数查询当前设置。如 `*pjsk服务器 jp` |
| `*抓包帮助` `*抓包` | 获取抓包上传教程 |
| `*抓包数据` `*pjsk check data` | 查看抓包数据在自建工具/Haruki工具箱中的更新时间 |
| `*抓包模式` `*pjsk data mode` | 修改抓包获取模式。`default` 仅本地找不到时用Haruki；`latest` 两者都取用最新的；`local` 仅本地；`haruki` 仅Haruki。推荐 `default`。 |
| `*隐藏抓包` `*展示抓包` | 隐藏/展示 Suite 抓包功能内的详细数据，每个服务器需分别设置 |
| `*pjsk验证` `*pjsk verify` | 验证游戏账号，部分功能需验证后使用 |
| `*pjsk验证列表` | 查看已验证的账号列表 |

**订阅功能**：`*pjsk订阅 <功能名>` 在当前群订阅指定功能；`*pjsk取消订阅 <功能名>` 取消订阅。不加功能名查看当前订阅和可订阅功能。

## 活动

**如何查询活动：**

- 查**单个活动**：活动ID(`123`)、负数索引(`-1` 倒数)、箱活缩写(`mnr1`)
- 查**多个活动**（参数混合，空格分隔）：团名(`mmj`)、角色名(`miku`)、箱活(`ick箱`)/混活(`混活`)、Bonus属性(`cool` `蓝`)、年份(`2025年`)、活动类型(`普活` `5v5` `wl`)
- `leak` 参数仅用于活动列表，筛未开始的活动

| 指令 | 功能 |
|------|------|
| `*活动` `*pjsk event` | 查单个活动。`*活动` 当期活动；`*活动 123`；`*活动 -1`；`*活动 mnr1` |
| `*活动列表` `*pjsk events` | 按条件筛选活动列表。`*活动列表 25h 蓝星 2025年`；`*活动列表 leak` 只看未开始；`*活动列表 mnr箱` |
| `*活动剧情` `*pjsk event story` | AI总结活动剧情。加 `refresh` 刷新总结。仅日服。 |
| `*送火` `*自动送火` | 5v5活动用游客号送3次免费火。仅日服。⚠️ 存在风险，自行承担。 |
| `*活动记录` `*冲榜记录` | 查询排名/PT前几的冲榜记录，需要📡抓包数据。数据增量保存。 |

## 组卡

使用📡抓包数据计算最优组卡。支持服务器：`所有`。

**组卡类型：**
- `*组卡` `*活动组卡` — 活动相关
- `*挑战组卡` — 每日挑战相关
- `*最强组卡` — 活动无关，分数/综合力/实效最大化
- `*加成组卡` `*控分组卡` — 凑加成控分
- `*烤森组卡` `*ms组卡` — 烤森获取PT（国服不开放）

## 烤森

支持服务器：`jp` `cn`。以下指令均需要📡烤森抓包数据（另有说明除外）。

| 指令 | 功能 |
|------|------|
| `*msd` `*烤森抓包` | 查看烤森抓包数据更新时间 |
| `*msr` `*mysekai资源` | 查询烤森资源（到访角色、资源一览、资源地图）。下午查询时早上到访角色无回忆图标。数据过期加 `force` 强制查询。 |
| `*msb` `*mysekai蓝图` | 查询蓝图持有情况。加角色昵称查家具对话收集（需同时有抓包数据和烤森数据）；附加 `all` 查看所有对话家具。`*msb mnr` |

| `*pjsk表情` `*pjsk stamp` | 获取或制作表情。`*pjsk表情 miku` 查角色表情列表；`*pjsk表情 miku 底图` 附带底图；`*pjsk表情 123` 获取透明底图片；`*pjsk表情 123 png` PNG格式；`*pjsk表情 123 你好世界` 制作DIY表情（支持换行） |

| `*pjsk表情刷新批量` | 🛠️ 为所有单人表情生成底图 |
| `*pjsk删除表情底图` | 🛠️ 删除AI生成底图 |
