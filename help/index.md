# 使用入门

NeneBot 为 Project Sekai 玩家提供多个 QQ Bot 协同服务。以下按功能分类列出简要可用指令与服务，具体指令请查看对应的文档。

使用帮助部分仍然可能存在表述问题和少许错误。

### 加群申请

如果你想将NeneBot拉到你的群里，可以去加群申请页面进行填写申请，审核通过后 再邀请Nene加入目标群就可以啦。

[前往加群申请 →](/apply)

## 基础指令

功能开发中 😴

### 帮助

| 指令 | 功能 |
|------|------|
| `help` `/help` `帮助` `/帮助` | 发送帮助信息 |

### Bot 开关

在群内按需启用/禁用指定 Bot，需群主或管理员权限。

| 指令 | 功能 |
|------|------|
| `/启用 <bot>` | 在当前群启用指定 Bot |
| `/禁用 <bot>` | 在当前群禁用指定 Bot |

### 发言榜

查询群内发言活跃度排名。

| 指令 | 功能 |
|------|------|
| `/发言榜` | 查看今日发言排行 |
| `/今日发言榜` | 同上 |
| `/总发言榜` | 查看历史总计发言排行 |
| `/发言榜 <日期>` | 查看指定日期发言排行。如 `/发言榜 -1`（昨日）、`/发言榜 -2`（前天）、`/发言榜 2024-01-15`（具体日期） |
| `/发言榜 <起始>~<结束>` | 查看指定日期区间发言排行。如 `/发言榜 -3~-1`（相对）、`/发言榜 2024-01-01~2024-01-15`（具体日期） |

## 世界计划 (PJSK)

### HarukiBot NEO

《世界计划 多彩舞台》多服查询 Bot，提供卡牌、歌曲、活动、榜线、组卡、烤森等完整 Sekai 查询服务。

前缀：`/` · 示例：`/绑定+id` `/song 虾` `/组卡 164`

[查看完整文档 →](/help/haruki)

---

### Sakura Bot

Project Sekai 查询 Bot，支持日/国/台服。群聊中直接使用指令，少数指令无法调用可加 `skr` 前缀强制调用。

指令直接触发 · 示例：`pjskb30` `cn个人信息` `组卡 ln 橙` `pjskcard miku`

[查看完整文档 →](/help/sakura)

---

### LunaBot — Sekai 查询

LunaBot 的 Sekai 游戏数据查询功能，覆盖账号设置、个人信息、歌曲谱面、卡牌组卡、活动榜线、烤森养成等模块。

前缀：`*` · 示例：`*pjskdetail` `*song 虾` `*查卡 miku`

[查看完整文档 →](/help/lunabot-sekai)

[查看完整文档 →](/help/lunabot)

### ZeroBot

插件机器人，基于 ZeroBot 框架。100+ 插件覆盖聊天、群管、娱乐、游戏、社交、工具、ACG、B站、MC 等类别。

控制插件触发方式：`@bot /` · 示例：`@bot /服务列表` `@bot /用法 qqwife`

[查看完整文档 →](/help/zerobot)
