# 常见问题

使用 NeneBot 过程中的常见疑问与解答。

<div class="faq-list">

<details class="faq-item">
<summary>如何上传 Suite / MySekai 数据？</summary>
<div class="faq-answer">
<p>iOS 用户可以使用 Surge、Loon 或 Shadowrocket 导入抓包模块；Android 用户可以在虚拟机中运行 catcher 抓包脚本；你也可以直接手动抓取文件到网页上传。具体配置见 <a href="/nenebot/upload/help">抓包配置帮助</a>。手动抓取需要到 <a href="/nenebot/upload/suite">Suite 上传</a> 或 <a href="/nenebot/upload/mysekai">MySekai 上传</a> 页面提交数据文件。</p>
</div>
</details>

<details class="faq-item">
<summary>Bot 没有响应我的指令？</summary>
<div class="faq-answer">
<p>常见原因：</p>
<ol>
  <li><strong>前缀错误</strong> — LunaBot 使用 <code>*</code>，HarukiBot NEO 使用 <code>/</code>。请确保前缀正确且有使用</li>
  <li><strong>指令路由过滤</strong> — Haruki 与 Sakura 的部分指令存在过滤机制，可能过滤不完善。HarukiBot NEO部分加上区服前缀可正常响应，Sakura Bot部分加 <code>skr</code> 前缀可强制路由到 Sakura</li>
  <li><strong>服务宕机啦</strong> </li>
</ol>
</div>
</details>

<details class="faq-item">
<summary>Haruki、Sakura、LunaBot 在世界计划部分的功能有什么区别？</summary>
<div class="faq-answer">
<p><strong>HarukiBot NEO</strong> — PJSK全服全功能Bot，功能多样且稳定，是最广泛使用的服务，以/作为前缀。</p>
<p><strong>Sakura</strong> — 提供部分HarukiBot NEO不支持的功能，如b30查询和民间定数，支持服务器稍少（jp/cn/tw）。</p>
<p><strong>LunaBot</strong> — 是个人部署的lunabot，仅支持日服和国服,功能全面，使用*作为前缀，与HarukiBot NEO功能重合度高，请自行查看和HarukiBot Neo的功能区别。</p>
</div>
</details>

<details class="faq-item">
<summary>如何申请 Bot 加入我的群？</summary>
<div class="faq-answer">
<p>到 <a href="/nenebot/apply">Bot 加群申请</a> 页填写群号、拉群目的和你的 QQ 号。平台会自动验证目标群聊是否真实存在，管理员审核通过后你需手动将 Bot 拉入群聊，Bot 会自动通过入群验证。</p>
</div>
</details>

</div>
